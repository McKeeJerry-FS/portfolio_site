This Is My Story...

I am a software developer based in Northeastern CT. I started back in 2019 by learning HTML, CSS, and JavaScript through self-studying means (coding sites like Codecademy, freeCodeCamp.org, etc...). In 2020 I decided to enroll in Full Sail University in the Web Design and Development program. There I was exposed to C# and .NET for the first time. I enrolled in a bootcamp with Woz-U and Infosys to build up my skills with .NET and SQL Server. After my first few jobs with >NET I realized I needed more training and in 2023 I enrolled in the Guardian Cohort of Coder Foundry. Here I returned to the beginning refreshing my knowledge of HTML, CSS, Bootstrap, and JavaScript as well as .NET and SQL Server. I spent the time building my confidence in my ability to create scallable programs and responsive websites. This portfolio is a collection of my works in programming. Take a look and see if we can work together on your next project.

Tech Set

C# has been the first language that I learned and the most passionate that I am about when it comes to learning. You can see my work implememnting C# in my projects section

I utilize ASP.NET Core and MVC to build scalable and interactive applications to satisfy client needs. Please check out my work in the projects section.

I use HTML, CSS, and Bootstrap to create beautiful and responsive webpage for any use. Frontend technologies are considered not as impresive to many other developers, but I embrace the technologiesas they are the very basic fundamentals of web design.

JavaScript is the language the brings responsive psages to life. I utilize Vanilla JS along with a host of other JavaScript frameworks to bring interactivity to my pages. Please check out my challenges section for interactive examples.

Persistence is achieved utilizing SQL. My projects utilize PostgreSQL on many of my projects, though I can write SQL on any DB platform

I use git and GitHub for all of my projects. Git is the most used source control framework on the planet. I employ CI/CD for publishing. The Javascript projects are hosted through Netlify, while the ASP.NET MVC applications are hosted via Railway along with an accompanying PostgresQL Database

Kontact Keeper Pro

Kontact Keeper Pro

Keep the contacts in your life organized with our new Kontact Keeper Pro. Create categories to organize your friends, family, colleagues, and so much more. Know that your data is safe with User Authentication provided by Microsoft and that your data is stored securely using the PostgreSQL 16 Database. Create your account and try it out today!

The project is built out using ASP.NET MVC and utilizes Microsoft Identity for User Authentication and Authorization. I utilized Entity Framework Core after creating the needed models to create the entities in a PostgreSQL Databse through migration. This makes this a Code-first Database project since I'm creating the models and relationships first, then through Migrations, create the tables within the database.

The project also utilizes an email service built out using MimeKit and MailKit add via nuget packages within the project. The Email Service first allows an email to be sent otu to verify email for registration. The other uses for the email service is to be able to send an email to a single contact or to be able to send emails out to a group of contacts based on the category that is chosen by the User.
